HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML memberikan struktur dasar dari konten di halaman web seperti teks, gambar, dan elemen lainnya. Dengan HTML, kita dapat menentukan elemen seperti header, paragraph, link, dan table yang memberikan struktur pada halaman web.

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web yang ditulis menggunakan HTML. Dengan CSS, kita bisa mengubah warna latar belakang, font, ukuran teks, dan bahkan mengatur tata letak elemen di halaman. CSS memungkinkan pengembang untuk membuat desain halaman web yang responsif dan menarik tanpa mengubah struktur HTML-nya.
